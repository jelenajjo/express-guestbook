## Instalation guide:

```
git clone https://github.com/jelenajjo/express-guestbook

cd express-guestbook

npm install
```

## Start app:

**!Make sure that there is no other app running on localhost:3000**

`node app.js`

Open http://localhost:3000 to preview app. :)
